SkimDoc

We are a software development outsourcing and outstaffing company

About the Client

SkimDoc is a cloud based application that allows to upload .pdf and other file format documents to the cloud, share links to them and see the document engagement statistics.

This kind of application is very useful for content marketers, business development management and PR professionals.

The document engagement analytics allows to see how your document if your document has been viewed, shared and how many times did these events occur.

The Challenge

The key challenge was to create an intuitive interface that would deliver the wide feature scope via a compact user interface.

Technologies used

Our Solution

We were required to create an intuitive web app design and create a set of mobile app prototypes for future product development.

The result of our work is an elegant user experience and user interface solution that incorporates simplicity and effectiveness.
